```java
public static int binarySearch(int[] arr, int target, int left, int right){
    if(left>right)
        return -1; //未找到
    int mid=left+(right-left)/2;
    if(arr[mid]==target)
        return mid;
    else if(target<arr[mid])
        return binarySearch(arr, target, left, mid-1);
    else
        return binarySearch(arr, target, mid+1, right);
}
```
